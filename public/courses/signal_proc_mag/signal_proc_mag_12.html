Статья

Задание:

Построить классификатор для китайских тонов на материале датасета Tone Perfect. Используйте любые из перечисленных признаков и/или их комбинации.

  1. вектор из N значений F0, выраженных в Гц, пт, эрбах, ...
  2. вектор из N значений первой, второй, третьей производной F0
  3. любые признаки, описанные в статье "Автоматическая обработка просодического оформления высказывания"
Конечная задача — сравнить эффективность признаков и построить сводную таблицу. Архитектура классификатора определяется совместно.
