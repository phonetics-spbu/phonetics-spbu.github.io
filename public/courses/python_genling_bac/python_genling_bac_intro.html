Домашние задания сдаются строго до дедлайна и присылаются на мою корпоративную почту в одном из трёх видов:

  • Файл .py (предпочтительно)
  • Файл .ipynb
  • Ссылка на Google Colab

При определённом проценте выполненных домашних заданий (будет определён позже) зачёт за семестр будет выставлен автоматом.

Зачёт подразумевает выполнение задания, аналогичного тем, которые были даны в течение семестра.

Инструкция по прикреплению файлов к письму

  1. В Проводнике в меню "Вид" установите флажок "Отображать расширения файлов"
  2. В своём почтовом клиенте кликните кнопку "Прикрепить файл"
  3. В открывшемся окне Проводника перейдите в папку, куда вы сохранили файл со своим кодом
  4. Выберите нужный файл. Убедитесь, что его имя заканчивается на расширение .py или .ipynb

Инструкция по отправлению ссылки на Google Colab

  1. Кликните по иконке "Поделиться" в верхнем правом углу
  2. В нижней части окна установите доступ по ссылке всем, у кого она есть, в качестве читателя (не редактора!)
  3. Скопируйте ссылку, нажав на кнопку в нижнем левом углу окна
  4. Вставьте ссылку в письмо
