На отрезке [-π, π] комплексное разложение непрерывной функции с периодом 2π имеет вид: $$f(t) = \sum_{k=-\infty}^{\infty}{C_k e^{jkt}}$$ $$C_k = \frac{1}{2\pi} \int_{-\pi}^{\pi}f(t)e^{-jkt}dt$$

Будем считать, что обрабатываемый нами сигнал дискретный и на рассматриваемом интервале задан N точками. Пусть интервал дискретизации равен Δt, тогда длительность данного отрезка сигнала - NΔt.

Дискретный сигнал является вектором из N элементов: $$x[n] = (x_0, x_1, ..., x_{N-1})$$ Чтобы получить коэффициенты Фурье-разложения дискретного сигнала x[n], нужно задать систему базисных векторов, соответствующую системе ортонормированных функций в непрерывном случае. Компонентами этих векторов будут выборки из соответствующих базисных функций с интервалом 2π/N: $$e^{jkt} \rightarrow \overline{e_k} = (1, e^{jk\Delta\omega}, e^{2jk\Delta\omega}, e^{3jk\Delta\omega}, ..., e^{(N - 1)jk\Delta\omega}), $$ $$ \Delta\omega = \frac{2\pi}{N}$$

В N-мерном векторном пространстве ряд векторов $$\{\overline{e_k}\} = \{\overline{e_0}, \overline{e_1}, \overline{e_2}, ..., \overline{e_{N-1}}\}$$ образует ортонормированный базис: $$ \langle \overline{e_m}, \overline{e_n} \rangle = \frac{1}{N} \sum_{i=0}^{N-1} e^{j\frac{2\pi}{N}mi}e^{-j\frac{2\pi}{N}ni} = \delta_{mn} $$

Тогда, разложение дискретного сигнала x[n] по данной системе базисных векторов будет выглядеть следующим образом: $$ x[n] = \sum_{k=0}^{N-1} X[k] e^{j\frac{2\pi}{N}kn} = \sum_{k=0}^{N-1} X[e^{j\omega}] e^{j\omega n}$$ $$\omega = \frac{2\pi}{N}k$$ Операция восстановления сигнала по известным коэффициентам разложения X[k] называется обратным дискретным преобразование Фурье (ОДПФ)) / (iDFT - inverse discrete Fourier transform).

Вычисление самих коэффициентов разложения сигнала называется (прямым) дискретным преобразование Фурье (ДПФ) / (DFT - discrete Fourier transform): $$ X[k] = \frac{1}{N} \sum_{n=0}^{N-1}x[n]e^{-j\frac{2\pi}{N}kn} = \frac{1}{N} \sum_{n=0}^{N-1}x[n]e^{-j\omega n} = X[e^{j\omega}]$$
