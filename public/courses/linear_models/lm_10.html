Дискретный сигнал

Сигнал - это то, что несет в себе какие-то данные. Сигнал можно представить как функцию одной или нескольких переменных (например, речевой сигнал - это функция времени). Переменная может быть как непрерывной, так и дискретной. В первом случае сигнал называется аналоговым. Дискретные сигналы зависят от переменной с дискретными значениями.

В этом курсе мы будем обсуждать сигналы с равномерной дискретизацией, то есть такие, у которых последовательные значения дискретной временной переменной находятся друг от друга на равных интервалах. Тогда сигнал можно представить в виде числовой последовательности (последовательности отсчетов) с известным шагом дискретизации T.

$$ x = \{x[n]\}, -{\infty} < n < {\infty} $$ где n - целое число, а {x[n]} - значение n-го отсчета дискретного сигнала.

Значение n-го члена последовательности равно величине соответствующего аналогового сигнала в момент времени nT, т.е. $$ x[n] = x_a(nT), -{\infty} < n < {\infty} $$ Число T называют шагом дискретизации, а обратное ему 1/T - частотой дискретизации.

Линейные системы

Пусть сигнал x[n] входит в некоторую систему и преобразуется в ней в сигнал y[n]. Последовательность x[n] называется входной последовательностью, а y[n] - реакцией или откликом системы. Обозначить это можно следующим образом: $$y[n] = T\{x[n]\}$$ Важно: значение n-го отсчета последовательности y[n] может зависеть от всех отсчетов последовательности x[n], либо от их части.

Пример системы: скользящее среднее. Она вычисляет n-й отсчет выходной последовательности как среднее арифметическое фиксированного количества отсчетов входной последовательности слева и справа от n-го отсчета входной последовательности вместе с самим отсчетом x[n]: $$y[n] = \frac{1}{M_1 + M_2 + 1} \sum_{k=-M_1}^{M_2}x[n-k] = \frac{1}{M_1 + M_2 + 1} (x[n+M_1] + x[n+M_1-1] + ... + x[n] + ... + x[n-M_2])$$

Система называется линейной, если соблюдается принцип суперпозиции: аддитивность $$T\{x_1[n] + x_2[n]\} = T\{x_1[n]\} + T\{x_2[n]\}$$ и однородность $$T\{ax[n]\} = aT\{x[n]\}$$ Одной формулой определение линейности системы можно записать так: $$T\{ax_1[n] + bx_2[n]\} = aT\{x_1[n]\} + bT\{x_2[n]\}$$

Задание: Докажите линейность скользящего среднего.

Пример нелинейной системы: $$y[n] = log_{10}|x[n]|$$ $$x_1[n] = 1 \Rightarrow y_1[n] = 0$$ $$x_2[n] = 10 \Rightarrow y_2[n] = 1$$ $$x_2[n] = 10x_1[n], y_2[n] \neq 10y_1[n]$$

Стационарные системы

Стационарные системы - это системы, у которых временной сдвиг входной последовательности вызывает соответствующий сдвиг выходной последовательности, то есть $$T\{x[n - n_0]\} = y[n-n_0]$$ Скользящее среднее - стационарная система.

Пример нестационарной системы: уплотнитель $$y[n] = x[Mn]$$ Что происходит: система отбрасывает M - 1 из каждых M отсчетов входного сигнала, оставляя только M-й.

Реакция на сдвиг по времени: $$y_1[n] = x[Mn - n_0] = x_1[Mn]$$ С другой стороны: $$y[n - n_0] = x[M(n - n_0)]$$ Но должно быть: $$y_1[n] = y[n - n_0]$$ Значит, уплотнитель - нестационарная система.

Линейные стационарные системы

В этом курсе далее мы рассматриваем только линейные стационарные системы.

Единичным импульсом называется последовательность вида: $$\delta[n] = \left\{ \begin{array}{rcl} 0, n \neq 0 \\ 1, n=0 \end{array}\right. $$ Любая последовательность может быть выражена в виде линейной комбинации единичных импульсов со сдвигом: $$x[n] = \sum_{k=-\infty}^{\infty}x[k]\delta[n-k]$$ Линейная система полностью определеяется своей реакцией на сдвинутые импульсные последовательности: $$y[n] = T\{\sum_{k=-\infty}^{\infty}x[k]\delta[n-k]\} = \sum_{k=-\infty}^{\infty}x[k]T\{\delta[n-k]\} = \sum_{k=-\infty}^{\infty}x[k]h_k[n], \text{ где } h_k[n] - \text { реакция системы на } \delta[n - k].$$ То есть, реакция линейной системы на любую входную последовательность выражается через отклики системы на сигнал δ[n-k].

Теперь добавим условие стационарности: если h[n] - реакция на δ[n] (импульсная характеристика системы), то ее реакция на δ[n-k] равна h[n - k]. Тогда, $$y[n] = \sum_{k=-\infty}^{\infty}x[k]h[n-k]$$ То есть, линейная стационарная система полностью определяется своей импульсной характеристикой h[n], в том смысле, что при известной последовательности h[n] можно вычислить любой отклик y[n].

Последовательность y[n] называют дискретной сверткой последовательностей h[n] и x[n]: $$y[n] = x[n]*h[n]$$, а уравнение $$y[n] = \sum_{k=-\infty}^{\infty}x[k]h[n-k]$$ - основным уравнением свертки. Для вычисления свертки последовательности x[k] и h[n-k] для каждого k перемножаются, а затем произведения суммируются по всем k; таким образом, получается отсчет выходной последовательности y[n].

Рассмотренная выше система скользящего среднего является линейной стационарной системой. Найдем ее импульсную характеристику. $$y[n] = \frac{1}{M_1 + M_2 + 1} \sum_{k=-M_1}^{M_2}x[n-k] = \sum_{k=-\infty}^{\infty}x[k]h[n-k]$$ Тогда, $$ h[n] = \frac{1}{M_1 + M_2 + 1}\sum_{k=-M_1}^{M_2}\delta[n-k] = \left\{ \begin{array}{rcl} \frac{1}{M_1 + M_2 + 1}, -M_1 \leq n \leq M_2 \\ 0, \text{ в остальных случаях} \end{array}\right.$$

Свойства линейных стационарных систем

Линейные системы описываются уравнением свертки, поэтому свойства этих систем определяются свойствами дискретной свертки. Следовательно, импульсная характеристика системы h[n] является полным описанием ее свойств.

Коммутативность $$x[n] * h[n] = h[n] * x[n]$$ или $$y[n] = \sum_{m=-\infty}^{\infty}x[n-m]h[m] = \sum_{m=-\infty}^{\infty}h[m]x[n-m]$$ Значит, отклик системы на сигнал x[n] при импульсной характеристике h[n] будет таким же, что и реакция на сигнал h[n] при импульсной характеристике x[n].

Дистрибутивность относительно сложения $$x[n] * (h_1[n] + h_2[n]) = x[n] * h_1[n] + x[n] * h_2[n]$$ (Соответствует параллельному соединению ЛС-систем)

Ассоциативность $$y[n] = (x[n] * h_1[n]) * h_2[n] = x[n] * (h_1[n] * h_2[n])$$ (Соответствует каскадному соединению ЛС-систем)
