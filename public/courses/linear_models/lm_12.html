Пара сигнал-отклик может быть связана линейным разностным уравнением порядка N: $$ \sum_{k=0}^{N}a_ky[n-k] = \sum_{m=0}^{M}b_mx[n-m] $$ При N = 1: $$ a_0y[n] + a_1y[n - 1] = \sum_{m=0}^{M}b_mx[n-m] $$ $$ a_0y[n] = \sum_{m=0}^{M}b_mx[n-m] - a_1y[n - 1] $$ То есть, каждое следующее значение выходного сигнала можно получить из предыдущего (или нескольких предыдущих при N > 1) и какого-то количества значений входного сигнала.

1) Cумматор описывается разностным методом достаточно просто: $$ y[n] = \sum_{k=-\infty}^{n}x[k] = x[n] + \sum_{k=-\infty}^{n-1}x[k] = x[n] + y[n-1] $$ То есть, для сумматора: $$ N = 1, a_0 = 1, a_1 = -1, M=0, b_0 = 1 $$

2) Рассмотрим скользящее среднее: $$ \text{Пусть } M_1 = 0 \text{ то есть система причинна - использует для вычисления отклика только значения до текущего отсчета входного сигнала. Тогда:} $$ $$ y[n] = \frac{1}{M_2 + 1} \sum_{k=0}^{M_2}x[n-k] = \frac{1}{M_2 + 1} (x[n] + x[n-1] + ... + x[m-M_2]) $$ $$ y[n - 1] = \frac{1}{M_2 + 1} \sum_{k=0}^{M_2}x[n- 1 -k] = \frac{1}{M_2 + 1} (x[n - 1] + x[n-2] + ... + x[m - 1-M_2]) $$ $$ y[n] - y[n-1] = \frac{1}{M_2 + 1} (x[n] - x[n-M_2 - 1]) $$ Значит, для скользящего среднего коэффициенты равны: $$ N = 1, a_0 = 1, a_1 = -1, M=M_2 + 1, $$ $$ b_0 = -b_{M_2 + 1} = \frac{1}{M_2 + 1}, b_k = 0 \text{ при } k\neq 0, k \neq M_2 + 1 $$
