Задание:

Написать функцию, которая принимает на вход слово на русском языке с отмеченным ударением (значок выберите сами), а возвращает его же, но записанное по принципу "как слышится, так и пишется". Какие явления нужно отразить:

  1. Редукция гласных:
    • молоко -> малако
    • лесной -> лисной
    • мясной -> мисной
    • шести -> шысти
  2. Оглушение в конце:
    • код -> кот
    • звёзд -> звёст
  3. Ассимиляция по звонкости:
    • отбор -> адбор
    • сделать -> зделать
    • безвкусный -> бисфкусный
    • партвзнос -> пардвзнос
    • женитьба -> жынидьба
    • дядька -> дятька
Если у вас есть сомнения по поводу того, как работает ассимиляция по звонкости в русском языке, см. сюда (стр. 31)
