Задание 1:

Написать функцию, которая принимает на вход слово в виде последовательности символов (кортеж строк) и алфавит символов, а возвращает все возможные слова, отстоящие от заданного на расстояние Левенштейна = 1: заменить по очереди все символы слова на все символы алфавита (замены), вставить по очереди во все позиции все символы алфавита (вставки), удалить по очереди все символы слова (удаления).

Задание 2:

Использовать функцию для создания словаря квазиомонимов (минимальных пар).
  1. Выбрать любой произносительный словарь из двух предложенных (см. материалы). Словарь lexicon.txt поменьше, но в нём есть ошибки. Словарь prondict.txt большой (и может долго обрабатываться), в нём ошибки менее очевидные, но его транскрипции написаны слитно, их надо дополнительно делить на звуки (знаки ударения и мягкости должны остаться приклеенными к предыдущим символам).
  2. Определить алфавит фонетических символов для выбранного словаря.
  3. Для транскрипции каждого слова (можно обрабатывать не весь словарь, а найти данные о частотности и взять только N самых частотных слов) сгенерировать потенциальные квазиомонимы и оставить только те, которые есть в словаре (на заметку: время поиска в dict или set гораздо меньше, чем в list).
  4. Определить классы квазиомонимов (см. пдфку в материалах).
  5. Построить графики по аналогии с пдфкой.
