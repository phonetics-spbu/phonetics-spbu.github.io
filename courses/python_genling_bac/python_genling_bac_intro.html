Домашние задания сдаются строго до дедлайна и присылаются на мою корпоративную почту в одном из трёх видов:

  • Файл .py (предпочтительно)
  • Файл .ipynb
  • Ссылка на Google Colab

При определённом проценте выполненных домашних заданий (будет определён позже) зачёт за семестр будет выставлен автоматом.

Зачёт подразумевает выполнение задания, аналогичного тем, которые были даны в течение семестра.
