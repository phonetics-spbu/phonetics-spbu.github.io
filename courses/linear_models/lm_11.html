Исходя из вида h[n] системы можно поделить на два вида: те, у которых импульсная характеристика по всем n конечна, и те, у которых она бесконечна. Например, рассмотрим систему "сумматор". В ней величина y[n] равна сумме всех предыдущих отсчетов до текущего n: $$y[n] = \sum_{k = -\infty}^{n}x[k]$$ $$h[n] = \sum_{k = -\infty}^{n}\delta[k] = \left\{\begin{array}{rcl} 0, n < 0 \\ 1, n \geq 0 \end{array}\right. = u[n]$$ Импульсная характеристика сумматора представляет собой функцию единичного скачка, она не ограничена по длительности. Системы такого рода называются системами с бесконечной импульсной характеристикой (БИХ-системы / IIR - infinite impulse response). В свою очередь, системы с конечным количеством ненулевых отсчетов называются системами с конечной импульсной характеристикой (КИХ-системы / FIR - finite impulse response).

Система устойчива, если и только если ее реакция на любой ограниченный импульс является ограниченной: $$\exists B_x: \forall n |x[n]| \leq B_x \text{ (ограниченная входная последовательность)} \Longleftrightarrow$$ $$\exists B_y: \forall n |y[n]| \leq B_y{ (ограниченная реакция)}$$ Рассмотрим отклик сумматора: $$y[n] = \sum_{k=-\infty}^{n}u[k] = \left\{\begin{array}{rcl} 0, n < 0 \\ n + 1, n \geq 0 \end{array}\right.$$ Сумматор не является устойчивой системой, т.к. для любого числа мы сможем найти достаточно большое n больше этого числа.

ЛС-система является устойчивой, тогда и только тогда, если ее импульсная характеристика - абсолютно суммируемая последовательность, т.е.: $$S = \sum_{k=-\infty}^{\infty}|h[k]| < \infty$$ Пример устойчивой БИХ-системы - система с испульсной характерисикой: $$h[n] = a^nu[n], |a| < 1$$ $$S = \sum_{k=-\infty}^{\infty}|h[k]| = \frac{1}{1 - |a|} < \infty$$

Фильтрация

Фильтрация - обработка сигнала во временной области, в результате которой изменяется спектральный состав исходного сигнала. Обычно эти изменения заключаются в ослаблении или подавлении некоторых нежелательных спектральных компонентов сигнала. Фильтры также бывают с бесконечной и конечной импульсной характеристикой.

КИХ-фильтры обычно (но необязательно) используют только текущий и какое-то конечное количество предыдущих отсчетов сигнала (тогда они называются нерекурсивными). Входная последовательность содержит конечное число ненулевых отсчетов, а значит и выходная послдеовательность содержит конечное число отсчетов конечной длительности.

Примеры КИХ-фильтров:

  • Скользящее среднее
  • Фильтр нижних частот (ФНЧ / low-pass filter) - пропускает спектр ниже некоторой частоты, подавляет все, что выше нее. ФНЧ бывает идеальным (подавляет всё) и неидеальным (гладкое подавление)
  • Фильтр верхних частот (ФНЧ / high-pass filter) - пропускает частоты выше какого-то порога. Идеальный ФВЧ физически нереализуем.

Рассмотрим систему скользящего среднего. Для вычисления n-го выходного отсчета нам фактически нужно поделить каждый отсчет из набора на одно и то же число, а затем их просуммировать. Коэффициенты, применяемые к отсчетам входной последовательности для получения выходного n-го отсчета, наызваются коэффициентами КИХ-фильтра. В случае с КИХ-фильтрами, коэффициенты фильтра равны его импульсной характеристике.

Теорема о свёртке

Вспомним теорему о свёртке: $$\mathscr{F} \{h(t)*x(t)\} = H(\omega) X(\omega)$$ То есть, дискретное преобразование Фурье (ДПФ) свёртки двух векторов во временной области равно их произведению в частотной области (и наоборот). Частотная характеристика фильтра: $$H(\omega) = \mathscr{F}\{h(t)\}$$ Итак, чтобы получить спектр выходного сигнала КИХ-фильтра, нужно перемножить спектр входного сигнала с частотной характеристикой фильтра.

Подытожим,

  • КИХ-фильтры выполняют операцию свёртки, суммируя произведения отсчётов сдвинутой входной последовательности на коэффициенты фильтра $$y[n] = \sum_{M_1}^{M_2}h[k]x[n-k]$$
  • Выходная последовательность КИХ-фильтра равна свёртке входной последовательности с импульсной характеристикой фильтра (последовательностью коэффициентов).
  • Частотная характеристика КИХ-фильтра равна ДПФ импульсной характеристики фильтра $$H(\omega) = \mathscr{F}\{h(t)\}$$
  • Спектр выходного сигнала КИХ-фильтра равен произведению спектра входного сигнала и частотной характеристики фильтра $$\mathscr{F}(y[n]) = H(\omega)X(\omega) = \mathscr{F}\{h(t)\}\mathscr{F}\{x(t)\}$$
  • Свёртка во временной области и произведение в частотной области связаны ДПФ. $$y[n] = \mathscr{F}^{-1}\{\mathscr{F}(y[n])\} = \mathscr{F}^{-1}\{H(\omega)X(\omega)\}$$
